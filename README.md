# SLD Screening Tool

A modern, comprehensive screening tool for Specific Learning Disabilities with a futuristic UI.

## Features

- 10 comprehensive assessment categories
- Student photo capture
- Professional PDF report generation
- Detailed analytics and visualizations
- Futuristic dark theme with particle animations

## Deployment

This is a static web application that can be deployed to any static hosting service.

### Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Tailwind CSS
- Chart.js
- jsPDF
- Particles.js
- Font Awesome

## Local Development

Simply open `index.html` in a modern web browser.
